# BAEKJOON ONLINE JUDGE - 1045. 도로

- [문제출처](https://www.acmicpc.net/problem/1045 '1045. 도로')

## 알고리즘 분류

- 그래프 이론
- 그리디 알고리즘
- 최소 스패닝 트리

## 풀이

### 접근

- `유니온 파인드`

### 설계

- 주어진 도로를 통해 유니온 파인드 수행
- 유니온 파인드를 수행하고 나서 모든 도로가 연결되어 있다면 `m`개의 도로를 사용할 때 까지 우선순위가 높은 도로를 추가로 연결
  - 도로가 연결되지 않거나 `m`개의 도로를 사용하지 못하는 경우 `-1` 출력
