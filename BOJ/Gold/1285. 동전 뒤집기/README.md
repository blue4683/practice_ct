# BAEKJOON ONLINE JUDGE - 1285. 동전 뒤집기

- [문제출처](https://www.acmicpc.net/problem/1285 '1285. 동전 뒤집기')

## 알고리즘 분류

- 그리디 알고리즘
- 브루트포스 알고리즘
- 비트마스킹

## 풀이

### 접근

- 완전 탐색

### 설계

- 행을 먼저 뒤집고 뒷면의 개수에 따라 열을 뒤집는 방식으로 탐색
  - 비트를 통해 행을 뒤집는 경우의 수 탐색 후, 열마다 뒤집을 때와 아닐 때의 개수를 비교하여 작은 수를 반영
  - 모든 경우의 수를 탐색하고 결과 출력
