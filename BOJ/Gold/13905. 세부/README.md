# BAEKJOON ONLINE JUDGE - 13905. 세부

- [문제출처](https://www.acmicpc.net/problem/13905 '13905. 세부')

## 알고리즘 분류

- 자료 구조
- 그래프 이론
- 그래프 탐색
- 최단 경로
- 분리 집합
- 최소 스패닝 트리

## 풀이

### 접근

- `유니온 파인드`

### 설계

- 무게 제한이 큰 순서대로 경로를 정렬하고 유니온 파인드 수행
- 수행하면서 출발 지점과 도착 지점이 연결되면 그 경로의 무게 제한을 결과값으로 업데이트
- 모든 경로를 탐색해도 연결되지 않는 경우 `0` 그대로 출력
