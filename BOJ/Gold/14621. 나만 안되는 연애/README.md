# BAEKJOON ONLINE JUDGE - 14621. 나만 안되는 연애

- [문제출처](https://www.acmicpc.net/problem/14621 '14621. 나만 안되는 연애')

## 알고리즘 분류

- 그래프 이론
- 최소 스패닝 트리

## 풀이

### 접근

- `유니온 파인드`

### 설계

- 길이가 작은 경로부터 확인하며 유니온 파인드 수행
  - 유니온이 되어있지 않으면서 대학교 종류가 다른 노드들만 유니온 수행
- 유니온 파인드 수행 후 모두 연결되어 있으면 경로 길이 출력, 아니라면 -1 출력
