# BAEKJOON ONLINE JUDGE - 14676. 영우는 사기꾼?

- [문제출처](https://www.acmicpc.net/problem/14676 '14676. 영우는 사기꾼?')

## 알고리즘 분류

- 구현
- 그래프 이론
- 방향 비순환 그래프

## 풀이

### 접근

- `구현`

### 설계

- 건물을 짓기 위한 선행 건물의 개수를 `indegree`에서 관리하고 지어진 건물의 개수를 `built`에서 관리
  - `indegree`가 0인 경우 건물을 지을 수 있음
  - 건물을 짓고 나서 처음 짓는 경우 그 건물이 선행 조건인 건물들의 `indegree`에서 1을 빼줌
