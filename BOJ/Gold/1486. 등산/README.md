# BAEKJOON ONLINE JUDGE - 1486. 등산

- [문제출처](https://www.acmicpc.net/problem/1486 '1486. 등산')

## 알고리즘 분류

- 그래프 이론
- 최단 경로
- 다익스트라
- 플로이드-워셜

## 풀이

### 접근

- `다익스트라`

### 설계

- 주어진 지도의 `(y, x)` 좌표를 노드 번호 `y * m + x`로 치환하여 그래프 생성
- 인접한 좌표에서 높이 차이가 `t`보다 크지 않은 좌표를 찾아 그래프에 소요시간과 함께 저장
  - 되돌아오는 경로도 찾기 위해 역으로 돌아오는 그래프를 함께 갱신
- 두 번의 다익스트라로 `0`번 노드에서 각 노드까지의 최단 소요 시간과 각 노드에서 `0`번 노드까지의 최단 소요 시간을 탐색
- `0`번 노드에서 각 노드까지 왕복하는데 소요되는 시간이 `d`를 넘지 않으면서 최대 높이를 찾아 출력
