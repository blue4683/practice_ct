# BAEKJOON ONLINE JUDGE - 15559. 내 선물을 받아줘

- [문제출처](https://www.acmicpc.net/problem/15559 '15559. 내 선물을 받아줘')

## 알고리즘 분류

- 자료 구조
- 그래프 이론
- 그래프 탐색
- 깊이 우선 탐색
- 분리 집합

## 풀이

### 접근

- `DFS`

### 설계

- DFS 수행
  - 현재 탐색하고 있는 사이클과 아닌 사이클을 분리하기 위해 상태값을 `0, 1, 2`로 판별
  - 탐색하고 있을 때 현재 사이클과 다시 만나면 결과값을 증가시키고 탐색된 사이클과 만나는 경우는 증가 X
