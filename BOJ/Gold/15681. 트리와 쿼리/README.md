# BAEKJOON ONLINE JUDGE - 15681. 트리와 쿼리

- [문제출처](https://www.acmicpc.net/problem/15681 '15681. 트리와 쿼리')

## 알고리즘 분류

- 다이나믹 프로그래밍
- 그래프 이론
- 그래프 탐색
- 트리
- 깊이 우선 탐색
- 트리에서의 다이나믹 프로그래밍

## 풀이

### 접근

- `DFS` + `DP`

### 설계

- 루트 노드를 시작으로 트리를 `DFS`로 탐색
  - 이미 탐색한 노드는 다시 탐색하지 않음
  - `dp` 배열에 서브 트리 정점의 수 갱신
