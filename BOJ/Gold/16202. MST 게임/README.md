# BAEKJOON ONLINE JUDGE - 16202. MST 게임

- [문제출처](https://www.acmicpc.net/problem/16202 '16202. MST 게임')

## 알고리즘 분류

- 그래프 이론
- 최소 스패닝 트리

## 풀이

### 접근

- `유니온 파인드`

### 설계

- 가중치를 내림차순으로 정렬하고 뒤에서부터 유니온 파인드 수행
- 모든 경로를 탐색했을 때 연결되었으면 최소 가중치를 결과값에 저장하고 연결되지 않으면 0 저장
  - 결과값 저장 후 가장 가중치가 작은 간선을 제거하고 반복
