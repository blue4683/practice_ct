# BAEKJOON ONLINE JUDGE - 16957. 체스판 위의 공

- [문제출처](https://www.acmicpc.net/problem/16957 '16957. 체스판 위의 공')

## 알고리즘 분류

- 다이나믹 프로그래밍
- 자료 구조
- 그래프 이론
- 그래프 탐색
- 분리 집합
- 격자 그래프

## 풀이

### 접근

- `유니온 파인드`

### 설계

- BFS로 모든 좌표를 탐색할 시 시간 초과 발생 -> 탐색 과정에서 중복 탐색하는 경우가 매우 많음
- `(y * c) + x`를 번호로 하는 그래프를 생성하고 `y, x` 주변만 탐색해 이동할 수 있는 좌표의 번호와 유니온 수행
  - 유니온을 수행할 때 `y, x` 위치의 값과 비교해서 업데이트
