# BAEKJOON ONLINE JUDGE - 16958. 텔레포트

- [문제출처](https://www.acmicpc.net/problem/16958 '16958. 텔레포트')

## 알고리즘 분류

- 그래프 이론
- 브루트포스
- 기하학
- 최단 경로
- 플로이드-워셜

## 풀이

### 접근

- `플로이드-워셜`

### 설계

- 주어진 도시 정보를 기반으로 그래프 생성
  - 텔레포트를 사용할 수 있는 경우 텔레포트 시간과 기본 이동 시간을 비교해 최솟값으로 반영
- 플로이드 워셜을 통해 각 도시별 최단 거리 갱신
