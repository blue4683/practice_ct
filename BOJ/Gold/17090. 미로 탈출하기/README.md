# BAEKJOON ONLINE JUDGE - 17090. 미로 탈출하기

- [문제출처](https://www.acmicpc.net/problem/17090 '17090. 미로 탈출하기')

## 알고리즘 분류

- 다이나믹 프로그래밍
- 그래프 이론
- 그래프 탐색
- 깊이 우선 탐색
- 격자 그래프

## 풀이

### 접근

- `DFS`

### 설계

- DFS를 수행하며 탈출할 수 있는지 없는지를 배열에 저장하며 탐색
  - 이미 방문했지만 탈출할 수 없는 노드라면 -1을 저장
  - 모든 노드를 탐색하고 탈출할 수 있는 노드의 개수를 출력
