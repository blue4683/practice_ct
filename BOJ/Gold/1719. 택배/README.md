# BAEKJOON ONLINE JUDGE - 1719. 택배

- [문제출처](https://www.acmicpc.net/problem/1719 '1719. 택배')

## 알고리즘 분류

- 그래프 이론
- 최단 경로
- 다익스트라
- 플로이드-워셜

## 풀이

### 접근

- `플로이드`

### 설계

- 경유 지점을 저장할 배열 생성
- 플로이드를 수행하면서 값이 업데이트 될 때 경유 경로를 저장
  - 값이 업데이트 될 때의 경유 지점을 저장할 경우 가장 먼저 이동해야하는 지점이 아닐 수 있으므로 `result[i][k]`을 저장
