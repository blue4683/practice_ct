# BAEKJOON ONLINE JUDGE - 1963. 소수 경로

- [문제출처](https://www.acmicpc.net/problem/1963 '1963. 소수 경로')

## 알고리즘 분류

- 수학
- 그래프 이론
- 그래프 탐색
- 정수론
- 너비 우선 탐색
- 소수 판정
- 에라토스테네스의 체

## 풀이

### 접근

- `BFS`

### 설계

- 10000이하의 수에서 소수인 수들을 체크하는 배열 생성
- BFS로 각 자리의 수를 변환하는 경우 탐색
  - 값이 1000보다 작아지거나 소수가 아닌 경우는 탐색 X
