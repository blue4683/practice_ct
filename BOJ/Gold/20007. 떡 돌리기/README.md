# BAEKJOON ONLINE JUDGE - 20007. 떡 돌리기

- [문제출처](https://www.acmicpc.net/problem/20007 '20007. 떡 돌리기')

## 알고리즘 분류

- 그래프 이론
- 정렬
- 최단 경로
- 다익스트라

## 풀이

### 접근

- `다익스트라`

### 설계

- 다익스트라로 `y`에서 최단 거리를 구하고 거리별로 정렬
- `최단 거리 * 2`가 `x` 이하가 아닌 집이 있다면 `-1`를 출력하고 아닌 경우 하루에 갈 수 있는 거리 안에서 최대한 집을 갈 수 있는 날짜를 탐색
