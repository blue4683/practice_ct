# BAEKJOON ONLINE JUDGE - 2110. 공유기 설치

* [문제출처](https://www.acmicpc.net/problem/2110 "2110. 공유기 설치")

## 알고리즘 분류
- 이분 탐색
- 매개 변수 탐색

## 풀이
### 접근
- 이분 탐색

### 설계
- 지금 위치가 기준 위치와 최소 거리이상만큼 거리가 있다면 카운트 증가
- 끝까지 탐색하고 카운트가 공유기 수 이상이라면 시작을 `mid+1`로 설정하고 결과값을 `mid`로 갱신
- 아니라면 끝을 `mid-1`로 설정
- 시작이 끝보다 커지면 이분 탐색 종료
