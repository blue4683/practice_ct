# BAEKJOON ONLINE JUDGE - 2293. 동전 1

- [문제출처](https://www.acmicpc.net/problem/2293 "2293. 동전 1")

## 알고리즘 분류

- 다이나믹 프로그래밍

## 풀이

### 접근

- `DP`

### 점화식

```python
dp[i]=dp[i-2]*3
for j in range(4,i+1,2):
    dp[i]+=dp[i-j]*2
```

### 설계

- 길이가 홀수일때는 채울수 없으므로 짝수일때만 확인한다.
- 길이가 `4`이상일때부터 고유의 패턴들이 등장한다.
    - `4`, `6`, `8` ... 모두 모양이 다르기 때문에 따로 더해주어야한다.
