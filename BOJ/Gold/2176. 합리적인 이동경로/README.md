# BAEKJOON ONLINE JUDGE - 2176. 합리적인 이동경로

- [문제출처](https://www.acmicpc.net/problem/2176 '2176. 합리적인 이동경로')

## 알고리즘 분류

- 다이나믹 프로그래밍
- 그래프 이론
- 최단 경로
- 다익스트라

## 풀이

### 접근

- `다익스트라 + DFS`

### 설계

- 다익스트라로 각 노드에서 `2`번 노드로 가는 최단 경로 탐색
- 1번 노드에서 출발해 2번 노드까지 가는 경로 중 `x -> 2`의 거리가 `xx -> 2`의 거리보다 큰 경우에만 탐색하고 이를 dp 배열에 반영
  - 경로 탐색을 DFS로 구하고 결과값 출력
