# BAEKJOON ONLINE JUDGE - 2307. 도로검문

- [문제출처](https://www.acmicpc.net/problem/2307 '2307. 도로검문')

## 알고리즘 분류

- 그래프 이론
- 최단 경로
- 다익스트라
- 역추적

## 풀이

### 접근

- `다익스트라` + `DFS`

### 설계

- 다익스트라로 최단 거리를 탐색하고 DFS로 최단 경로를 탐색
- 최단 경로에서 도로를 막는 경우의 최단 거리를 탐색하고 막지 않는 경우와의 차이의 최댓값을 출력
  - 도로를 막아 도착할 수 없는 경우 `-1`을 출력하고 종료
