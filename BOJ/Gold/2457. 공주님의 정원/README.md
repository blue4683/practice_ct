# BAEKJOON ONLINE JUDGE - 2457. 공주님의 정원

- [문제출처](https://www.acmicpc.net/problem/2457 '2457. 공주님의 정원')

## 알고리즘 분류

- 그리디 알고리즘
- 정렬

## 풀이

### 접근

- 구현

### 설계

- 달에 100을 곱해 일과 더해준 값을 사용하고, 배열에 저장한 뒤 정렬
- 끝나는 날짜를 `301`로 초기화해주고 꽃 중에서 끝나는 날짜보다 먼저피고 가장 늦게 지는 꽃을 탐색
- 끝나는 날짜가 `1201` 이상이 되는 경우 또는 그러한 꽃이 없는 경우 탐색을 종료하고 결과값 출력
