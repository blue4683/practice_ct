# BAEKJOON ONLINE JUDGE - 2585. 경비행기

- [문제출처](https://www.acmicpc.net/problem/2585 '2585. 경비행기')

## 알고리즘 분류

- 그래프 이론
- 그래프 탐색
- 이분 탐색
- 너비 우선 탐색
- 매개 변수 탐색

## 풀이

### 접근

- `이분 탐색` + `BFS`

### 설계

- 연료통의 용량을 기준으로 이분탐색 수행
  - 이분탐색의 매개변수를 연료통의 용량으로 목적지까지 도착할 수 있는 지의 여부로 사용
  - BFS로 목적지까지 갈 수 있는지 탐색
