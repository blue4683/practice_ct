# BAEKJOON ONLINE JUDGE - 2611. 자동차경주

- [문제출처](https://www.acmicpc.net/problem/2611 '2611. 자동차경주')

## 알고리즘 분류

- 다이나믹 프로그래밍
- 그래프 이론
- 방향 비순환 그래프
- 위상 정렬

## 풀이

### 접근

- `위상 정렬`

### 설계

- `1`로 돌아오는 경로를 제외한 경로들로 그래프 생성
- 각 경로들의 차수를 저장해 `2 ~ n`번 지점으로 갈 수 있는 경로 중 최대 점수를 `dists`에 갱신하며 탐색
  - `parent` 배열로 최대 점수를 얻을 때의 이전 노드를 저장
- `1`에서 출발해 `i`번으로 도착했을 때의 최대 점수와 `i`번에서 `1`으로 도착하는 경로의 점수를 더해 최댓값을 갱신
- `parent` 배열을 통해 경로 생성
