# BAEKJOON ONLINE JUDGE - 2613. 숫자구슬

- [문제출처](https://www.acmicpc.net/problem/2613 '2613. 숫자구슬')

## 알고리즘 분류

- 다이나믹 프로그래밍
- 그리디 알고리즘
- 이분 탐색
- 매개 변수 탐색
- 역추적

## 풀이

### 접근

- `이분 탐색`

### 설계

- 각 그룹의 합의 최댓값을 이분 탐색으로 구함
  - 중간값이 넘어갈때까지 그룹에 포함시키고, 가능한지 확인
- 그룹의 개수가 `m`보다 작은 경우 그룹의 크기가 `1`보다 큰 그룹을 반으로 쪼개면서 `m`개까지 확장
