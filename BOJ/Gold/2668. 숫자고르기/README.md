# BAEKJOON ONLINE JUDGE - 2668. 숫자고르기

- [문제출처](https://www.acmicpc.net/problem/2668 '2668. 숫자고르기')

## 알고리즘 분류

- 그래프 이론
- 그래프 탐색
- 깊이 우선 탐색

## 풀이

### 접근

- `DFS`

### 설계

- 인덱스에 해당하는 값을 인덱스로 하여 순회하면서 사이클이 생기는지 확인
  - 사이클이 생기면 결과값에 반영
    - 기존 결과값과 겹치는 인덱스가 없다면 결과값에 그냥 더해줌
    - 기존 결과값과 겹친다면 더 큰 조합을 결과로 갱신
