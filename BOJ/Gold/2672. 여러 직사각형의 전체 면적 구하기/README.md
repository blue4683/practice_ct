# BAEKJOON ONLINE JUDGE - 2672. 여러 직사각형의 전체 면적 구하기

- [문제출처](https://www.acmicpc.net/problem/2672 '2672. 여러 직사각형의 전체 면적 구하기')

## 알고리즘 분류

- 스위핑

## 풀이

### 접근

- `구현`

### 설계

- `x` 좌표를 기준으로 `y` 의 증감을 통해 면적을 구함
- `x` 좌표를 이동시켰을 때 덮여진 `y`의 길이를 곱하고 현재 `x` 좌표에서 `y` 구간의 증감을 반영
- 결과 출력 시 `int` 타입일 경우 `int`로 출력하고 아닐 경우 소숫점 2번째 자리까지 출력
