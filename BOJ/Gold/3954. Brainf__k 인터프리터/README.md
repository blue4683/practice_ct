# BAEKJOON ONLINE JUDGE - 3954. Brainf\*\*k 인터프리터

## 알고리즘 분류

- 구현
- 자료 구조
- 시뮬레이션
- 스택

## 풀이

### 접근

- `구현`

### 설계

- 루프 인덱스를 매칭시키기 위해 스택에 `[]`의 인덱스를 저장하고 꺼내면서 딕셔너리에 저장
- 주어진 조건에 따라 구현
  - `cnt`가 5천만을 넘어갈때부터 무한 루프일때를 대비해 루프의 시작 인덱스를 찾아 저장
