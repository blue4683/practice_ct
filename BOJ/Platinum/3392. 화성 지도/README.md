# BAEKJOON ONLINE JUDGE - 3392. 화성 지도

- [문제출처](https://www.acmicpc.net/problem/3392 '3392. 화성 지도')

## 알고리즘 분류

- 자료 구조
- 세그먼트 트리
- 스위핑

## 풀이

### 접근

- `세그먼트 트리`

### 설계

- `x` 좌표를 기준으로 `y` 의 증감을 통해 면적을 구함
- `x` 좌표를 이동시켰을 때 덮여진 `y`의 길이를 곱하고 현재 `x` 좌표에서 `y` 구간의 증감을 반영
- 직사각형의 개수가 많아 시간 복잡도를 줄이기 위해 세그먼트 트리로 현재 구간에 직사각형의 개수와 구간 길이를 저장
