# Programmers - Level 2. 광물 캐기

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/172927 'Level 2. 광물 캐기')

## 풀이

### 접근

- `그리디`

### 설계

- 다이아몬드, 철, 돌에 점수를 매기고 5개마다의 점수의 합을 기준으로 점수가 가장 높은 순으로 좋은 곡괭이를 사용해 채굴
  - 철 5개보다 다이아몬드 1개, 돌 5개보다 철 1개의 가중치를 높이기 위해 이전 광물의 점수의 5배보다 높게 점수를 매김
  - 곡괭이를 다 써도 광물을 모두 캘 수 없는 경우가 있으므로 점수를 매기는 광물의 수는 곡괭이의 수의 5배와 광물의 수 중 작은 수를 기준으로 진행
