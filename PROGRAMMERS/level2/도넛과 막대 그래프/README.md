# Programmers - Level 2. 도넛과 막대 그래프

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/258711 'Level 2. 도넛과 막대 그래프')

## 풀이

### 접근

- `구현`

### 설계

- 노드로 진입하는 간선은 없고 진출만 하는 간선이 2개 이상이라면 임의로 추가한 노드
- 노드로 진입하는 간선이 있고 진출하는 간선이 2개 이상이라면 8자 그래프
- 노드로 진입하는 간선은 있고 진출하는 간선은 없으면 막대 그래프
- 도넛 그래프는 임의로 추가한 노드의 진출하는 간선의 개수에서 8자 그래프의 개수와 막대 그래프를 뺀 개수와 같음
