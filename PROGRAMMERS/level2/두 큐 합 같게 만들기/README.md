# Programmers - Level 2. 두 큐 합 같게 만들기

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/118667 'Level 2. 두 큐 합 같게 만들기')

## 풀이

### 접근

- `슬라이딩 윈도우`

### 설계

- 두 큐를 이어붙인 배열을 생성
  - 각 큐의 원소의 합이 홀수라면 절대 같아질 수 없으므로 `-1` 출력
- 범위를 첫번째 큐로 지정하고 각 큐의 원소의 합의 절반 값을 기준으로 범위를 갱신
- 더이상 범위를 늘리거나 줄일 수 없거나 기준값에 도달하면 반복문 종료 후 결과값 출력
