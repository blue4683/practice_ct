# Programmers - Level 2. 리코쳇 로봇

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/169199 'Level 2. 리코쳇 로봇')

## 풀이

### 접근

- `BFS`

### 설계

- 상하좌우로 탐색을 수행하며 보드 밖으로 나가기 전 또는 장애물을 만나기 전까지 같은 방향으로 이동
- 멈추었을때의 위치가 여태까지 방향을 바꾼 횟수보다 큰 경우에 이를 visited 배열에 기록하고 큐에 위치를 추가
- 목적지에 도착하는 경우 바꾼 횟수를 반환하고 모든 경우를 탐색해도 도착하지 못하는 경우 -1를 반환
