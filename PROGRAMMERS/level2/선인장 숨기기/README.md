# Programmers - Level 2. 선인장 숨기기

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/468379 'Level 2. 선인장 숨기기')

## 풀이

### 접근

- `슬라이딩 윈도우`

### 설계

- `deque`을 활용해 슬라이딩 윈도우를 구현
  - 윈도우에 `(위치, 값)`을 저장하고 윈도우가 오름차순으로 정렬되도록 원소 삽입
  - 가장 앞에 있는 값이 구간을 벗어나는 경우 `popleft`로 제거
- 행을 기준으로 슬라이딩 윈도우를 통해 구간 내의 최솟값을 탐색
- 위에서 구한 최솟값들로 열을 기준으로 하는 구간 내의 최솟값을 위와 같은 방법으로 탐색
