# Programmers - Level 2. 유사 칸토어 비트열

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/148652 'Level 2. 유사 칸토어 비트열')

## 풀이

### 접근

- `구현`

### 설계

- 유사 칸토어 비트열은 무조건 5로 나눠짐(0과 1을 5자리로 교체하기 때문에)
- 현재 위치 인덱스를 5로 나누면 무조건 `0`(0과 1 둘다 교체되는 비트열의 두번째 자리가 `0`)
- 그 외는 시작값이 `0`일 경우 무조건 `0`(0과의 거리가 2인 경우는 `00000` 밖에 없음)
