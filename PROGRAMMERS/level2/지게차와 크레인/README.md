# Programmers - Level 2. 지게차와 크레인

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/388353 'Level 2. 지게차와 크레인')

## 풀이

### 접근

- `BFS`

### 설계

- `storage`를 `0`으로 감싸서 활용
  - `0`은 바깥과 연결되어 있고, `1`은 비어있는 의미
- 지게차와 크레인으로 컨테이너를 꺼내는 것을 `BFS`로 구현
- 지게차로 컨테이너를 꺼낼땐 무조건 바깥과 연결되어 있으므로 `0`으로 갱신하고, 크레인의 경우 아닐 수도 있기 때문에 `1`로 갱신
- 컨테이너를 꺼낸 뒤에는 무조건 `check` 함수로 바깥과 연결되어 있는 `1`을 `0`으로 갱신하는 작업을 수행
- 이를 반복하고 남아있는 컨테이너의 수를 출력
