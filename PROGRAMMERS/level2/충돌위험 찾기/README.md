# Programmers - Level 2. 충돌위험 찾기

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/340211 'Level 2. 충돌위험 찾기')

## 풀이

### 접근

- `구현`

### 설계

- 주어진 경로를 이동하도록 구현
  - 동시에 이동하는 로봇을 고려하기 위해 항상 새로운 큐를 만들어 사용하고 모두 이동해 더이상 큐에 아무것도 없을 경우 종료
  - 큐에 현재 위치와 목적지를 저장
    - 목적지가 여러개일 수 있으므로 주어진 포인트들을 역순으로 나열하고 목적지에 도착할때마다 `pop`을 통해 도착한 목적지를 제거
  - 이미 어떤 로봇이 이동했고, 아직 위험 상황임을 모르는 경우 결과값을 `1` 증가시키고 위험 상황을 저장하는 `set`에 저장하여 중복 고려를 방지
