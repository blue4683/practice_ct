# Programmers - Level 3. 표현 가능한 이진트리

- [문제출처](https://school.programmers.co.kr/learn/courses/30/lessons/150367 'Level 3. 표현 가능한 이진트리')

## 풀이

### 접근

- `구현`

### 설계

- 주어진 십진수를 이진수로 변환하고 포화이진트리의 개수와 동일하도록 `0`을 앞에 붙여줌
- 루트 노드에서부터 탐색하면서 루트 노드가 `0`이면서 자식 노드들 중 하나라도 `1`이 있는 경우는 불가능으로 판별하고 결과값을 리스트에 저장
